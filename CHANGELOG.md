# Changelog

All notable changes to this project will be documented in this file.

## [2.0.1] - 2026-02-08

### Changed
- Version bump to 2.0.1.

## [2.0.0] - 2026-02-07

### Added
- Major Feature Update.
- Fixed GTK theme parser warnings.
- Database reset improvements.
- UI strict color validation.

## [1.0.2] - 2026-02-07

### Added
- Database export functionality in Settings.
- Database import functionality in Settings.
- **Optional password encryption** for database backups.
- Confirmation dialogs for database reset and import.

### Fixed
- Fixed a bug where redundant UI elements were added to the Settings dialog.
- Improved UI responsiveness after database import.

## [1.0.1] - 2026-02-05

### Added
- Initial release of the Review Spaced Repetition Study Manager.
- Core scheduling logic (7, 15, 30 days).
- Study timer with fullscreen support.
- Calendar views (monthly/weekly).
- Topic management by areas and tags.
